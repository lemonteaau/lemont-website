# Terry Cheng Personal Website

My personal website built with Next.js.

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- React Three Fiber
- GSAP
- Framer Motion
- Shadcn/ui

## Quick Start

### Install Dependencies

```bash
pnpm install
```

### Development Server

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to view the website.

### Build for Production

```bash
pnpm build
pnpm start
```
